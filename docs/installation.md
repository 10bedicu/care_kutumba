# Installation

## Stable release

To install care_kutumba, run this command in your terminal:

```sh
pip install care_kutumba
```

## From source

The source files for care_kutumba can be downloaded from the [Github repo](https://github.com/yash-learner/care_kutumba).

You can either clone the public repository:

```sh
git clone git://github.com/yash-learner/care_kutumba
```

Or download the [tarball](https://github.com/yash-learner/care_kutumba/tarball/master):

```sh
curl -OJL https://github.com/yash-learner/care_kutumba/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd care_kutumba
pip install .
```
